# Área de Gestión de Tecnologías de Información (GTI)

**Competencia general (CE01):** Gestión e Innovación de TI — diagnostica brechas, prioriza iniciativas, justifica inversiones y orienta la tecnología hacia resultados organizacionales, aplicando el modelo CDIO (Conceive, Design, Implement, Operate) y estándares como PMBOK, Gobierno de TI y Gestión de Procesos.

## Metodología del diagnóstico

El diagnóstico se desarrolla bajo un enfoque de doble lente:

1. **Análisis interno** — evalúa a LOGYX como organización (el sujeto que ejecuta el proyecto)
2. **Análisis externo** — evalúa el ecosistema logístico PYME–transportista del corredor Lima–Sierra Sur, representado mediante una **PYME arquetipo** para el análisis de procesos (desarrollado en E4)
